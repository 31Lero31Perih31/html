O que é HTML?

Bom, é uma ótima ideia começar explicando o que HTML não é. Pois HTML não é uma linguagem de progra­mação. HTML é uma sigla em inglês para: HyperText Markup Language, ou seja, HTML é uma Lingua­gem de Marcação de HiperTexto. Ele é usado para definir uma estrutura em sites.
