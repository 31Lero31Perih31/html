O que é HTML?

O que é HTML?

Bom, é uma ótima ideia começar explicando o que HTML não é. Pois HTML não é uma lingua­gem de progra­mação. HTML é uma sigla em inglês para: HyperText Markup Language, ou seja, HTML é uma Lingua­gem de Marcação de HiperTexto. O que significa que ele é usado na criação de sites para definir estruturas como paragráfos, títulos, conteudo principal etc.

<!-- Resumo, HTML é uma linguagem de marcação de texto -->

Tags HTML

Tags Informações Sobre
<header> É o cabeçalho do seu site, geralmente contendo links e o título principal do site.
<main> É o conteudo principal do seu site, nele estarão todas as suas ideias e suas menssagens.
<footer> Esse é o rodapé do seu site, geral­mente contem créditos ou alguns links.
<h1>...<h6> São títulos, o número ao lado do h repre­senta seu grau de hierarquia.
<p> Esse é o parágrafo do seu site, use ele para criar textos.
